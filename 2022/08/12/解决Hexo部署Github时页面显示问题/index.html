解决Hexo部署Github时页面显示问题

发表于 2022-08-12 阅读次数：
部署 hexo d 时成功，但博客仍显示此前页面未更新

检查 _config.yml 文件：

发现不知道什么时候改为了 hexo ，文件都上传到 hexo 目录下去了，修改为 master 即可。

然后执行命令 hexo clean、hexo d -g尝试，这样就成功解决了。
